Run the controller.py as it is the starting point of the program.
When a option appears, please select the number.
Select the option of either 
1. Add ingredients to the pizza
You can choose either choose from cheese or toppings

If you select 'Cheese', you can choose from the following cheese:
* Parmigiano Reggiano
* Fresh Mozarella
* Vegan Cheese

If you select 'Topping', you can choose from the following topping:
* Peppers
* Pineapple
* Mushrooms
* Fresh Basil
* Spinach
* Pepperoni
* Beyond Meat

2. By selecting the 'pay' option, you can finish the transaction by getting
the included ingredients and the total cost
